Number System Definitions

Binary: A number system that uses only two digits, 0 and 1. It's the foundation of computer systems.

Octal: A base-8 number system that uses digits from 0 to 7. It is often used in digital systems.

Hexadecimal: A base-16 number system that uses digits 0–9 and letters A–F. Commonly used in programming.

Decimal: The standard base-10 number system we use every day, consisting of digits 0–9.
